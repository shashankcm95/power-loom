### Sections (per existing template)

Each KB includes:

1. Frontmatter (`kb_id` + `version` + `tags` + `sources_consulted` + `related` + `status`)
2. `## Summary` (5-6 bullets condensing key points)
3. `## Quick Reference` (canonical principles + examples + rules of thumb)
4. `## Intent` (why the principle exists)
5. Topic-specific deep sections (`## The Principle`, `## Recognizing X`, etc.)
6. `## Apply when` (decision criteria)
7. `## Substrate applications` (grounding examples)
8. `## Tensions` (counter-positions + tradeoffs)
9. `## History` (authoring batch + pairing rationale)
10. `## Phase` (H.9.3 attribution)

### Source citations

Each KB cites 4-5 sources from training-data familiarity. Citation discipline:

- Specific book + chapter (e.g., "Ousterhout, PoSD 2nd ed 2021, ch 5") not vague references
- arXiv papers cited with ID (e.g., `arXiv:2210.03629`) where applicable
- Anthropic / OpenAI documentation cited as named docs (anthropic.com/research, etc.)
- Avoid manufactured citations — if a specific paper claim is uncertain, use general references
